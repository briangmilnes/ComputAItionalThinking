# ComputAItionalThinking

project-tz: America/Los_Angeles

 This project is building a CLAUDE.md and Computer science vocabulary to encourage claude
 agents (and others) to think and speak in computer science terms when working on coding
 and verification tasks. And to avoid analogies and metaphors which make understanding
 the agents outputs slow and error prone.

The agent ruleset lives in `rules.md`, imported below so it is active when
working in this repo. Any other project may adopt the same ruleset by adding
`@~/projects/ComputAItionalThinking/rules.md` to its own `CLAUDE.md`.
