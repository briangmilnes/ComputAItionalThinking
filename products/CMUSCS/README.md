# CMU SCS — Course Vocabularies

Extracted vocabularies from Carnegie Mellon SCS courses whose most recent
offering has publicly readable lecture slides or notes (no login). Each course
has a directory here with its downloaded `materials/` and an extracted
`vocabulary.md` (term → 1–2 sentence description).

Source class list and slide/notes availability: `../../docs/CMUSCS/classlist.md`.
