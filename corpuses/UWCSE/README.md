# UW CSE — Course Vocabularies

Extracted vocabularies from University of Washington CSE courses whose most
recent offering has publicly readable lecture slides or notes (no NetID). Each
open course has a directory here with its downloaded `materials/` and an
extracted `vocabulary.md` (term → 1–2 sentence description).

Source class list and slide/notes availability: `../../docs/UWCSE/classlist.md`.
